Heart Disease Analysis with SQL & Power BI

Overview

This project explores heart disease risk factors using SQL queries and visualizes key insights with Power BI.

SQL Query Example

SELECT 
    HeartDisease, 
    AVG(Cholesterol) AS AvgCholesterol  
FROM heart_failure_staging  
GROUP BY HeartDisease;
